### 本サービスの概要（700文字以内）
本サービスは、家事の「忘れ」「不公平感」「感謝の伝わらなさ」を解消するための、パートナー向け家事サポートアプリです。家事の時間になると同居メンバー全員にスマホ通知が届き、通知に表示される「やります」ボタンを押すだけで家事への参加が完了します。アプリを開く必要がないため、日常生活の中で無理なく家事を分担できます。

参加した人にはポイント（ショートケーキ）が付与され、8個集まるとホールケーキに自動変換されます。貯めたホールケーキは「自分が欲しいご褒美」を相手にお願いするために使えます。頑張った人が頑張った分だけ評価され、自然と「ありがとう」が増える仕組みです。

想定ユーザーは、①家事の分担を曖昧にしたくない同棲カップル、②家事の不満を減らしたい夫婦、③忙しくて家事を見える化したい共働き家庭などです。「今余裕がある人がやる」という現実的な運用に合っており、家事の偏りを防ぎつつ、協力しやすい状態をつくります。

主な機能は、家事の自動リマインド、通知からのワンタップ参加、ポイント管理、ご褒美リクエスト、家事履歴表示、プロフィールの管理など。家事をゲーム感覚で楽しく続けられる“みんなで家事習慣を育てるアプリ”です。
